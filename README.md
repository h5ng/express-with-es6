# express-with-es6
express + es6


## Environment
- NPM 6.9.0
- Node 10.15.3

## Installation
```bash
$ npm i
```

## Usage

```bash
# 개발서버 실행
$ npm run watch

# 배포파일 생성 및 서버 실행
$ npm run start

# 테스트 실행
$ npm run test
```
